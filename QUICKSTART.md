# Quick Start Guide

## Installation Steps

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **For Android**
   ```bash
   npm run android
   ```

3. **For iOS**
   ```bash
   cd ios && pod install && cd ..
   npm run ios
   ```
